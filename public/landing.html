How it works

From care note to safer home visit.

AI identifies one or more tasks from the care note, then TaskBridge automates assignment using safeguarding, availability, price, proximity, and service fit.

Care note submitted

A coordinator logs concerns such as lawn mowing, window cleaning, loose rails, locks, or trip hazards.

AI creates tasks

One note can become multiple structured tasks with category, urgency, and preferred time window.

Safeguarding applied

Vulnerable adult tasks prefer Enhanced DBS-approved handymen or supervised visits when a carer is on site.

Assignment automated

Care teams see only pending assignment or assigned handyman details, never the full proposal list.

Safeguarding first

Built for vulnerable-adult protection.
